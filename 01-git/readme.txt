
01) Set the username and email for the local repository. This will override the global setting. 

git config user.name "Ravi Gadela"
git config user.email "ravi.gadela@gmail.com"

02) Commit Commands

Add all the modified files. 
git add . 

Commit the changes to local repository
git commit -m "commit message here"

Push the changes to remote repository
git push -u origin main 

03)Remote Branches Link

List of all the remote branches 
git branch -r

Local and Remote Tracking Branches
git branch -vv
